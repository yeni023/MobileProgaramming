* 로그인 화면, 마이페이지, 버전 정보, 로그아웃 화면 설계
* 안드로이드 스튜디오로 앱 개발
* 앱 로고 만들기
* 최종보고서 작성
* 앱 시연 및 앱 시연 영상 촬영
<br>

2. **팀원 김혜진**
* 로그인 화면, 일반 회원 로그인 화면, 관리자 로그인 화면, 메인 화면, 관리자, 도움말 화면 설계

* 앱 로고 만들기
* 최종보고서 작성
* 발표 자료 만들기
<br>

3. **팀원 박예은**
